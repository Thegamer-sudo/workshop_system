# Workshop Management System


## Overview
The Workshop Management System is a web-based application designed to manage and streamline student-led workshops. It allows users to register, view available workshops, and manage their participation efficiently.

## Features
- User registration and login
- Browse available workshops
- Register for workshops
- Admin management of workshops
- Track workshop participation
- Provide feedback about workshops

##  Technologies Used
- FLASK
- HTML, CSS, JavaScript
- SQL Database

## How to Run the Project
1. Clone the repository:
   git clone [https://github.com/Thegamer-sudo/workshop_system.git]

2. Open the project in your development environment (e.g., Visual Studio Code )

3. Set up the database (if applicable)

4. Run the application

Project Structure
- Controllers: Handles application logic
- Models: Represents data and database structure
- Views: User interface components
